手机里装着 Anki，还有 Duolingo，你也真心打算好好用它们。可一拿起手机，二十分钟后你已经在刷抖音了，都不太清楚自己是怎么点进去的。

ProcrastiLearn 就在这一刻出手拦截。就在你即将一头扎进又一个时间黑洞之前，它先问你一个单词。

于是，习惯性打开某个 App 这件事，就变成了一次词汇练习。挑出那些你总是无意识点开的应用——社交软件、浏览器、游戏——之后每次打开它们，屏幕上都会先弹出一张全屏卡片。看看翻译，评估一下自己记得怎么样，你要打开的应用马上就会正常启动。不用订阅，没有广告，也不用注册账号。


使用方法

在"应用"页面选择要拦截的 App。从此以后，打开这些 App 之前都会先看到一张卡片。你给出评价（"重来" / "困难" / "良好" / "简单"），FSRS 间隔重复算法会自动安排下一次复习时间。每日复习上限可以避免卡片堆积得让你喘不过气。

如果你一进入被拦截的 App 就容易忘了时间，还可以打开循环提醒功能：设置一个时间间隔（比如每 3 分钟），只要你还留在那个 App 里，卡片就会不断重新出现——原本会被浪费掉的时间，就这样变成了额外的复习机会。


词汇管理

可以手动添加单词，也可以交给 AI：粘贴一个词，点一下"生成"，应用就会调用 OpenAI 给出翻译。提示词完全可以自己编辑，方便调整语气、上下文，或者换成别的目标语言。

也支持直接导入已有的 Anki 词库（.apkg 文件），用你已经攒好的词表马上就能开始。


语言支持

目前应用对英语-俄语这一对语言支持最好（AI 提示词默认就是这一对），不过任意语言对都能用——只要去设置里把提示词改成你需要的语言就行。针对其他语言对的专属界面会在后续版本中逐步加入，尤其会参考大家在 GitHub issue 或邮件里提出的需求。


隐私

所有数据——词汇、被拦截的应用列表、复习记录、设置——都只保存在你的设备本地。没有数据分析，没有崩溃报告，没有广告 SDK。唯一会向外发送的网络请求，是你自己点击"生成"时触发的 OpenAI 调用，使用的是你在设置里填写的个人 API 密钥。


权限说明

无障碍服务：仅用于识别当前在前台运行的是哪个 App，这样 ProcrastiLearn 才知道什么时候该弹出卡片。该服务不会读取屏幕内容、按键记录或任何其他输入信息。

悬浮窗权限（显示在其他应用上层）：用于把卡片显示在被拦截的应用上方，没有这个权限卡片就画不出来。


开源

ProcrastiLearn 是一款免费的开源软件（Apache 2.0 协议）。源代码和问题追踪都在 GitHub 上。
